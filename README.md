# Development
### Clone this repo
1. `git clone git@github.com:SiliconJungles/smoothops.asia.git`
2. `bundle install`

### Start local dev server
1. `middleman`

### Build the web pack for deployment
1. `middleman build`

# Deployment
1. `middleman deploy`
